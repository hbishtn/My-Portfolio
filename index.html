Multi-Agent AI System (Online)

Cloud-hosted multi-agent AI platform built with LangChain & LangGraph for agent orchestration. Implements a full RAG pipeline backed by ChromaDB for vector storage and embedding-based retrieval. Celery & Redis handle async task queuing; Django serves as the web backend with PostgreSQL for persistence. Fully containerized with Docker. Includes a PyGUI desktop interface and Playwright-powered browser automation for real-time web interaction.
